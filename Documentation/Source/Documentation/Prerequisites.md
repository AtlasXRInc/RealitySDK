# Prerequisites

## Unreal engine
- Install Unreal Engine 5 <https://www.unrealengine.com/fr/download>

## Reality plugin
- Download latest Reality plugin <https://github.com/AtlasXRInc/Reality/releases> 
- Install it to `Plugins` folder of your project

## Beyond dev version
- Join the Beyond dev channel (by the invite link)
- Configure the channel on the quest
